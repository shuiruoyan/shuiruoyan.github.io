重点：其中于运算与短路与的区别？

与运算（&）是在运算符两边计算完成之后，再进行与运算，当在按位计算时表示按位与。
短路与是先计算左边的表达式，如果左边的表达式结果为false，则不计算右边的计算式，返回flase；如果左边的表达式结果为true，则会计算右边表达式，如果右边表达式结果也为true，返回结果true。

4. 位运算符

  1. |：按位或：有1则为1；
  2. &：按位与；有0则为0；
  3. ^：按位异或；不同为true，相同为false；
  4. ~：按位非：1变0，0变1。

5. 位移运算符

  1. a<<b：左移，将二进制形式的a逐位左移b位，低位补0；
  2. a>>b:带符号右移，将二进制形式的a逐位右移b位，高位空出的符号位按原符号位补齐；
  3. a>>>b：无符号右移，将二进制形式的a逐位右移b位，高位空出的符号位补0。

重点：左移操作在乘法中间的应用
